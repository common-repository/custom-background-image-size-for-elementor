== Description ==
Change the background-image size in an Elementor Container widget. It will display a smaller image in the frontend (editor will still show the full size).

How to use it:
* add a container
* set the background image
* change the 'Background size' (in the 'Style' tab at the end of 'Background' )

== Installation ==
* install Elementor
* install the plugin
* add a container
* set the background image
* change the 'Background size' (in the 'Style' tab at the end of 'Background' )
